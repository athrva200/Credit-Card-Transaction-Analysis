# Credit Card Transaction Analysis | Power BI

## Project Overview
Interactive dashboard analyzing credit card transaction data 
for 10,108 customers across 2023.

## Key Metrics
- Total Revenue: $7.84M
- Total Transaction Amount: $44.52M
- Total Transactions: 655,651
- Delinquency Rate: 6.1%

## Dashboard Features
- Revenue trend by quarter (Q3 highest at $2.04M)
- Spend breakdown by expense type (Bills top category)
- Card category performance (Blue dominates at $6.49M)
- Weekly transaction trend
- Interactive quarter slicer

## Tools Used
- Power BI Desktop
- DAX (Data Analysis Expressions)
- Data Modelling (Star Schema)

## Dataset
Source: Kaggle — Credit Card Customer Data
Rows: 10,108 | Tables: 2 (customer + credit_card)
